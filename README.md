# 📅 Wall Calendar Planner

An interactive and responsive wall calendar built using React.
It allows users to select date ranges and attach notes, combining a clean UI with practical functionality.

---

## 🚀 Features

* 📆 **Calendar View**

  * Monthly layout inspired by a physical wall calendar
  * Smooth navigation between months

* 🔵 **Date Range Selection**

  * Select start and end dates
  * Visual highlight for selected range

* 📝 **Notes Section**

  * Add notes for selected dates or ranges
  * Simple and intuitive interface

* 📱 **Responsive Design**

  * Works seamlessly on desktop and mobile devices

---

## 🛠 Tech Stack

* React (Vite)
* TypeScript
* Tailwind CSS
* date-fns (for date handling)

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/aasthamewar/Wall-Calendar.git
cd Wall-Calendar
```

2. Install dependencies:

```bash
npm install
```

3. Run the project:

```bash
npm run dev
```
